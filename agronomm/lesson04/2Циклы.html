2.Циклы с использованием for


     Одним из вариантов организации циклов и наиболее популярным является использование оператора for. Он обычно применяется в том случае, когда программист заранее знает, сколько раз будут выполняться относящиеся к циклу инструкции.

Синтаксис



for (Initialization; Condition; Modification)
{
Instruction(s)
}


  • Initialization (Инициализация): этот код выполняется один раз в самом начале исполнения данной инструкции. При дальнейшем выполнении инструкций цикла данная часть кода уже не имеет значения;
  • Condition (Условие): цикл и связанные с ним инструкции выполняются до тех пор, пока верно данное условие. Если условие не выполняется, повторение цикла прекращается;
  • Modification (Изменение, модификация): данная команда выполняется после очередного выполнения цикла и перед проверкой выполнения условия Condition.

Как контролируется выполнение цикла?



     Три части - Initialization; Condition; Modification –вместе образуют так называемый заголовок цикла. Он содержит команды, которые определяют, как часто будет выполняться цикл (то есть блок инструкций, который относится к данному циклу).
    Как мы можем контролировать частоту выполнения цикла?
    Очень просто – с помощью созданных для этой цели переменных, которые играют роль счетчика выполнения цикла.


Обычно эту переменную называют переменной цикла. В наших примерах эта переменная чаще всего носит название loop, но имя не играет никакой роли. Вы можете называть эту переменную исходя из своих соображении.
В самом начале выполнения цикла, в секции Initialization, переменной цикла присваивается начальное значение. При каждом выполнении блока инструкций этого цикла с помощью секции Modification изменяется значение переменной цикла. После того, как переменная цикла достигает некой величины, при которой уже не выполняется условие цикла, выполнение цикла прекращается.


Вычисление последовательности значений



     В качестве упражнения напишем программу, которая использует цикл. В данной программе должны вычисляться квадраты для десяти первых чисел.
1.Начнем новую программу ForCycle.java и создадим следующий каркас программы: 
for (int i = 0; i < 10; i++) {
}
Данный фрагмент кода начинается с объявления переменной цикла, которая в данном случае носит имя i. Перед началом исполнения инструкций цикла необходимо присвоить переменной цикла начальное значение, в данном примере для этого используется команда i=0;.
Далее с помощью i<=10 сформулировано условие для продолжения выполнения цикла. Условие Condition гласит, что цикл выполняется до тех пор, пока переменная цикла i имеет значение меньше или равное 10.
Третья инструкция выполняется каждый раз при окончании выполнения блока инструкций Instructions данного цикла. Видно, что после десяти выполнений данного цикла переменная i становится больше 10 и, следовательно, выполнение цикла прекращается.


Последняя инструкция в заголовке цикла не должна заканчиваться точкой с запятой. Также не должна использоваться точка с запятой и после круглой скобки, которая стоий в конце заголовка цикла.
3.Циклы с использованием while

Синтаксис



     При использовании оператора while стадии инициализации, проверки условия выполнения цикла и изменения не встроены в заголовок цикла, а должны быть явно выполнены в программном коде. Данный вариант чаще всего используется тогда, когда программист не знает заранее, сколько раз будет выполнен данный цикл.
Оператор while имеет следующий синтаксис: 

Initialization;
while (Condition)
{
Instruction(s); Modification;
}

Оператор switch

Оператор switch() удобно использовать в случае необходимости множественного выбора. Выбор осуществляется на основе целочисленного значения.

Структура оператора:

switch(int value) {
case const1:
выражение или блок
case const2:
выражение или блок
case constn:
выражение или блок
default:
выражение или блок
}

Причем, фраза default не является обязательной.
В качестве параметра switch может использоваться переменная типа byte, short, int, char или выражение. Выражение должно в конечном итоге возвращать параметр одного из указанных ранее типов. В операторе switch не могут применяться значения примитивного типа long и ссылочных типов Long, String, Integer, Byte и т.д.

При выполнении оператора switch производится последовательное сравнение значения x с константами, указанными после case, и в случае совпадения выполняется выражение следующее за этим условием. Если выражение выполнено нормально и нет преждевременного его завершения, то производится выполнение для последующих case. Если же выражение, следующее за case, завершилось ненормально, то будет прекращено выполнение всего оператора switch.

Если не выполнен ни один оператор case, то выполнится оператор default, если он имеется в данном switch. Если оператора default нет и ни одно из условий case не выполнено, то ни одно из выражений switch также выполнено не будет.

Следует обратить внимание, что, в отличие от многозвенного if-else, если какое-либо условие case выполнено, то выполнение switch не прекратится, а будут выполняться следующие за ним выражения. Если этого необходимо избежать, то после кода следующего за оператором case используется оператор break, прерывающий дальнейшее выполнение оператора switch.

После оператора case должен следовать литерал, который может быть интерпретирован как 32-битовое целое значение. Здесь не могут применяться выражения и переменные, если они не являются final static.


4.Модификация хода выполнения цикла

Существуют команды, которые позволяют:
  • Прекращать выполнение одного повторения цикла;
  • Прекращать выполнение всего оператора цикла.

Изменитть ход выполнения цикла


     Зачем нужно прерывать и останавливать выполнения цикла? Во многих циклах, наряду с переменной цикла, используются и другие переменные. При выполнении блока инструкций цикла эти переменные могут принимать значения, которые не позволяют продолжать нормальное выполнение цикла. Или же с самого начала может быть указано, что для определенных значении переменной выполнение цикла останавливается.
    Давайте представим, что необходимо вычислить квадраты для первых десяти чисел за исключением числа 5. Этого нельзя добиться с помощью инструкций заголовка цикла, однако существует возможность создать два различных цикла.
    Имеется полезная инструкция continue, которая нарушает обычное выполнение цикла и идеально подходит для нашей задачи.

1.Начнем новую программу ForCycle2.java и пренесем в нее текст программы ForCycle.java. Не забудьте при этом изменить название класса на ForCycle2:

{

{

{

Очень важно при использовании инструкции continue обеспечить правильную модификацию (увеличение или уменьшение) переменной цикла. В цикле, основанном на операторе for, данная проблема не возникает, так как при прерывании выполнения блока инструкции с помощью continue автоматически выполняется третья команда в заголовке цикла, которая осуществляет модификацию переменной. Однако в циклах с использованием while модификация переменной цикла должна проводится одной из команд, входящих в блок инструкций цикла! Таким образом, в циклах while необходимо размещать команду continue только после операции, которая производит модификацию переменной цикла. В противном случае, после первого вызова команды continue оператор цикла будет выполнятся бесконечно!

Остановить выполнение цикла


     Инструкция break очень похожа на инструкцию continue. Однако при ее вызове происходит прерывание не отдельной итерации выполнения цикла, а выполнение самого оператора цикла.
Данная инструкция используется в основном при решении задач следующего типа: “Выполнять данные инструкции до тех пор, пока не произойдет определенное событие”.
    Данную постановку задачи можно проиллюстрировать с помощью программы для вычисления квадратов значений из раздела “Циклы, которыми управляет пользователь” данной главы: “Считывать вводимые пользователем значения и вычислять для них квадрат до тех пор, пока пользователь не введет значение 0 ”.
    Попробуем модернизировать эту программу с помощью бесконечного цикла и инструкции break:
1.Создадим новую программу BreakDemo.java и перенесем в нее код из программы WhileCycle.java. 
//Вычисление квадратов чисел с помощью цикла while
import java.lang.*;
import.java.io.*;
public class BreakDemo
{
public static void main (String [] args) throws IOException
{
double inputVar = 1;
String inputString;

BufferedReader t = new BufferedReader (new InputStreamReader(System.in));
while (true)
{
System.out.print("Input number (0= end program): ");
inputString = t.readLine();
inputVar = integer.parseInt(inputString);
if (inputVar == 0)
break
System.out.println("Square" + inputVar + "is" + (inputVar * inputVar));
}
}
}


Вместо условия выполнения цикла мы ставим просто значение true. Компилятор оценивает, верно ли условное выражение с помощью булевских логических значений true или false. Указывая значение true, мы указываем компилятору, что условие цикла всегда верно и цикл будет выполняться дальше. 



После считывания значения, которое ввел пользователь, мы используем инструкцию if и сравниваем введенное значение с 0:
  • если значение не равно 0, то выполнение цикла продолжается;
  • если значение равно 0, управление передается инструкции break и выполнение оператора цикла прекращается.
