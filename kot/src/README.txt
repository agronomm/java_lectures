В программировании код в файлах с расширением java называется исходным, по 
английски sources или сокращенно src(сырцы или серцы в русском 
произношении). Соответственно весь код хранится в одноименной 
папке src. Дополнительные материалы(скриншоты, описания, доп файлики, и 
т.д.) класть в папку resources или аналогичные.
